# Task Manager - React + Express Full-Stack Application

A full-stack task management web application built using React and Express.  
This project demonstrates REST API integration, user authentication, and database-driven task management.

---

## Tech Stack

- Frontend: React 18, Vite, Axios  
- Backend: Node.js, Express  
- Database: SQLite (sql.js)  
- Authentication: bcryptjs  
- Styling: CSS3, Flexbox, Media Queries
